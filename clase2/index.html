Ejercicio 1

  1. El tercer parámetro recibido es una función. Dicha función es ejecutada luego de realizar realizar la suma los argumentos a y b
  2. Este tipo de funciones se deben utilizar en situaciones donde se trabaja con operaciones asíncronas. El callback permite especificar qué debe suceder después de que se haya completado una operación, como una llamada a la base de datos.

  3. Un ejemplo puede ser:

    suma(2,3, () => console.log("Suma completada"))
